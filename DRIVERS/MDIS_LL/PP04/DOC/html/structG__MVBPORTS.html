Detailed Description

the user application has to pass the NSDB to the LL driver with the MVB_BLK_NSDB Block SetStat. The M_SG_BLOCK data block has to consist of n continuous MVB_LA_PORT structs in memory. For example, the User App may define 10 ports with MVB_LA_PORT myMVBports[10]; and pass these as shown in the pp04_simp program
